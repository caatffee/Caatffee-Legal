A diferencia de otros bots, Caatffee prioriza la **privacidad del Staff**. Casi todo se gestiona por Mensaje Directo (MD):

  • Comando /help: Tu guía rápida de uso.

  • Eventos: Usa /create-event, /edit-event y /delete-event.

Preguntas Frecuentes

¿Por qué usar MD para configurar?
Para asegurar la privacidad de los administradores. Nadie en los canales públicos sabrá cómo estás configurando el bot.
¿Quién puede usar el bot?
Solo el propietario del servidor y los miembros con permisos de Administrador pueden ejecutar cambios.
¿Cómo edito un mensaje enviado?
Al solicitar una edición desde el MD, el bot borrará el mensaje anterior y enviará el nuevo corregido.

Buzón de Sugerencias

Términos de Servicio

1. Uso Aceptable

El usuario se compromete a no usar el bot para spam o acciones maliciosas. Las funciones administrativas son exclusivas para Administradores y Propietarios.

2. Servicio Premium

Los pagos son para mantenimiento y mejora. !No son reembolsables¡. No se permite el mal uso aunque seas Premium.

3. Responsabilidad

Caatffee no se hace responsable del mal uso de los usuarios (ej. mensajes ofensivos), aunque el bot tiene sistemas de prevención.

4. Gestión de Mensajes

Para ediciones, el bot borrará el contenido previo para asegurar una limpieza total en el canal de destino.

5. Modificaciones

Nos reservamos el derecho de cambiar estos términos para proteger la estabilidad del sistema.

Política de Privacidad

1. Recopilación de ID

Se recopila el ID de usuario para gestionar el funcionamiento y permisos del bot.

2. Datos Temporales

Una vez configurada una acción, el bot no guarda datos, excepto en eventos cronometrados donde la info es necesaria hasta la fecha del evento.

3. Terceros

Tus datos jamás serán compartidos con servicios de terceros o extraños.

4. Control Total

Puedes sacar al bot en cualquier momento. Los cambios hechos se quedan, pero el bot borrará toda tu información de su base de datos.

5. Privacidad MD

Toda la interacción de configuración ocurre en privado entre el usuario y el bot.
